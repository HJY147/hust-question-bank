# Question Images Directory

## Purpose
This directory stores **question images only** for the search system.

## File Naming Rules
- Use format: `subject_number.jpg`
- Examples:
  - `complex_003.jpg` - Complex Analysis Question 3
  - `calc_001.jpg` - Calculus Question 1
  - `phys_002.jpg` - Physics Question 2
  - `circuit_005.jpg` - Circuit Analysis Question 5

## Supported Formats
- `.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`

## Answer Files
- Each question image should have a corresponding answer file
- Answer location: `data/answers/question_id.txt`
- Example: `complex_003.jpg` → `data/answers/complex_003.txt`

## Important
⚠️ **DO NOT** put personal photos or landscape images here!
⚠️ Only upload question/problem images for the question bank.
